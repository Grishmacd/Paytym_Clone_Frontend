# Paytym_Clone_Frontend

A Paytm homepage UI clone built using HTML and CSS. This project recreates the layout and visual structure of Paytm’s landing page sections like the header navigation, UPI banner, recharge cards, travel booking UI, feature sections, and footer. It is a frontend-only clone created for learning layout building, positioning, and UI styling using pure CSS.

Disclaimer: This project is for educational/portfolio use only. Paytm is a registered trademark of One97 Communications Ltd. This project is not affiliated with Paytm.

## Project Overview
This project focuses on recreating Paytm-like interface sections such as:
- Header with logo, navigation menu, download button, and sign-in button
- UPI statement banner with download CTA
- Recharge & Bill Payments section with product icons
- Offer banners and promotional strips
- Paytm Travel section with flight search UI (trip type, from/to, depart/return, passenger/class, special fares)
- User payment showcase section (UI mock layout)
- Paytm Money / MTF promotion block
- Paytm for Business section
- Footer links and social icons

This is a UI clone made to practice HTML structure and CSS layout skills.


## Technologies Used

- HTML5 (structure)
- CSS3 (styling, layout, positioning)

## Features
- Paytm-style navbar with menu items and download image button
- UPI banner card with CTA button
- Recharge & bills grid with icons
- Promotional banner strips (cashback + broadband)
- Travel booking UI section with input layout and fare buttons
- Multi-card sections with large promotional images
- Footer with expandable-style link rows + social icons
- Built with pure HTML/CSS (no frameworks)
